This directory contains a JavaSound plug-in that allows resampling of audio. It was copied from http://www.tritonus.org/plugins.html, to have an archive in case that web site ever becomes unavailable. You'll probably want to go to http://www.tritonus.org/plugins.html and get the latest version.

According to http://www.tritonus.org/, these appear to be covered by the GNU Library General Public License.
